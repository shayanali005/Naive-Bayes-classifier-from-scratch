# Naive-Bayes-classifier-from-scratch
Naive Bayes Classifier: Implementation & AnalysisThis repository contains a comprehensive technical guide and implementation of the Naive Bayes algorithm. The core content is provided in a detailed PDF format, covering the mathematical theory, algorithmic logic, and practical code application.📌 Project OverviewNaive Bayes is a probabilistic machine learning model used for classification tasks. It is based on the Bayes' Theorem, which describes the probability of an event based on prior knowledge of conditions that might be related to the event.
$$P(y|X) = \frac{P(X|y)P(y)}{P(X)}$$
